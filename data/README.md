# Data included in the repository

`structural_examples/` contains small, selected structures used to document the
physical-relaxation panel. Reference structures originate from the COVID
Moonshot structural resource; generated and minimized structures are project
outputs.

The repository MIT License applies to original software, not automatically to
third-party structural data. Reuse reference structures under the terms of their
source resource and cite the associated dataset.

The complete generated campaign and large validation outputs are not stored in
GitHub. See `docs/DATA_LAYOUT.md`.

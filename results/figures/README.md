# Manuscript figure exports

This directory contains the current PDF/PNG/SVG/TIFF manuscript outputs.
Figures 1–4 and the pair-compatibility supplement can be regenerated from the
tracked inputs with `make figures`.

Figure 5 is archived here and can be reassembled after generating its standalone
PyMOL renderings with `make pymol-figure5-renderings`.

The quantitative figures currently reflect the frozen manuscript analysis.
They should be regenerated after the corrected campaign re-audit is completed.
